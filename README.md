# php_hazimetenoPHP

> PHP入門学習時に作成したWebアプリのサンプルコード集

## 概要

「はじめてのPHP」として入門書やチュートリアルを参考にしながら作成したWebアプリのサンプルです。
フォーム入力・検索・一覧表示など、基本的なPHPの機能を練習したものです。

- フォームからニックネーム・メールアドレス・意見を送信する機能
- 入力チェック・リスト表示・検索機能など

## ⚠️ 注意

このリポジトリは学習・練習目的で作成したものです。コードに誤りや古い書き方が含まれている可能性があります。動作は保証しません。

- HTML
